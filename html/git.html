Git

Создание репозитория

  • git init [название проекта] - создание нового локального репозитория с заданным именем.
  • git clone [url-адрес] - скачивание репозитория вместе со всей его историей изменений.

Внесение изменений

  • git status - перечисление всех новых или изменённых файлов, которые нуждаются в фиксации.
  • git add [файл] - индексация указанного файла для последующего коммита.
  • git commit -m "[сообщение с описанием]" - фиксация проиндексированных изменений и сохранение их в истории версий.

Просмотр истории

  • git log - отображение истории коммитов для текущей ветки.

Синхронизация с удаленным репозиторием

  • git push [удалённый репозиторий] [ветка] - загрузка всех изменений локальной ветки в удалённый репозиторий.
  • git pull - загрузка истории из удалённого репозитория и объединение её с локальной (pull = fetch + merge).
